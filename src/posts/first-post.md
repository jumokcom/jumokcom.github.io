# Next.js로 블로그 만들기

Next.js와 TypeScript를 사용하여 개발 블로그를 만들어보았습니다.
이 블로그는 다음과 같은 기술 스택으로 구성되어 있습니다:

- Next.js 14
- TypeScript
- Tailwind CSS
- Markdown 지원

## 왜 Next.js인가?

1. 뛰어난 성능
2. TypeScript 지원
3. 정적 생성과 서버 사이드 렌더링
4. 편리한 라우팅
5. 마크다운 지원

앞으로 이 블로그를 통해 다양한 개발 경험을 공유하겠습니다.
