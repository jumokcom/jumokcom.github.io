# 블로그를 시작하며

안녕하세요! 제 개발 블로그에 오신 것을 환영합니다.
이 블로그는 Next.js와 TypeScript를 사용하여 만들었습니다.

## 블로그를 시작하게 된 이유

개발을 하면서 배운 것들을 기록하고 공유하고 싶어서 블로그를 시작하게 되었습니다.
주로 다음과 같은 내용들을 다룰 예정입니다:

1. 웹 개발 이야기
2. 프로그래밍 팁
3. 새로운 기술 학습 기록
4. 프로젝트 회고

## 사용한 기술 스택

이 블로그는 다음과 같은 기술들로 만들어졌습니다:

- Next.js 14
- TypeScript
- Tailwind CSS
- Markdown

## 앞으로의 계획

앞으로 다음과 같은 기능들을 추가할 예정입니다:

- 댓글 시스템
- 검색 기능
- 카테고리/태그 기능
- 다크 모드

많은 관심과 피드백 부탁드립니다! 😊
